# Tutorials

Tutorials are learning-oriented. They guide you through a complete, working task to build understanding through doing. Each tutorial produces a verifiable outcome.

| Tutorial | What You Will Build |
| --- | --- |
| [Hello World: Your First API Call](hello-world.md) | Authenticated GET request returning live repository metadata |
| [Resource Interlinking](resource-interlinking.md) | A chained workflow traversing Repos, Issues, and Search |
| [Security Hardening](security-hardening.md) | Automating secret management and repository security auditing |

Unlike [How-to Guides](../how-to/index.md), tutorials do not assume you know which endpoint to call. They explain the mental model alongside the mechanics.
